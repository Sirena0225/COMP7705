## ✨ 核心特性

### 1️⃣ 实时数据处理

✅ **特点**:
- 处理真实流入的数据，而非离线静态数据集
- 支持每秒数十个样本的处理
- 自动采样和批量处理

✅ **采样策略** (4种):
```
UNIFORM     - 均匀采样（固定比例）
STRATIFIED  - 分层采样（按语言/来源均衡）❌ 推荐
ADAPTIVE    - 自适应采样（根据置信度）
RISK_AWARE  - 风险感知采样（高风险优先）
```

### 2️⃣ A/B并行测试

✅ **特点**:
- 生产模型和候选模型同时运行
- 自动计算一致性评分 (0-100)
- 性能对比分析

✅ **对比维度**:
```
极性一致性    - 40% 权重
风险等级一致  - 30% 权重
情绪分接近度  - 20% 权重
置信度接近度  - 10% 权重
```

### 3️⃣ 持续评估

✅ **评估指标**:
```
准确率 (Accuracy)
精确率 (Precision)
召回率 (Recall)
F1分数
延迟 (Latency) - mean, median, p95, p99
吞吐量 (Throughput)
一致性 (Consistency)
置信度 (Confidence)
```

✅ **时间序列分析**:
- 按时间间隔聚合
- 趋势分析
- 异常检测

### 4️⃣ 自动反馈和优化

✅ **自动调整**:
```
评估结果 → 分析 → 生成调整操作
    ↓
    ├─ 准确率 < 85% → 提示词优化
    ├─ 置信度 < 70% → 阈值调整
    ├─ 延迟 > 1000ms → 采样策略调整
    └─ 一致性 < 80% → 创建标注任务
```

✅ **提示词优化**:
- 自动分析错误模式
- 生成改进建议
- 动态更新提示词

### 5️⃣ 人机协同标注

✅ **Streamlit应用**:
- 交互式标注界面
- 优先级队列管理
- 实时统计仪表板
- 批量任务导入

✅ **标注工作流**:
```
检测需求 → 创建任务 → Streamlit展示
  ↓           ↓           ↓
优先级     按难度     标注者复核
队列       排序       生成标准
  ↓           ↓           ↓
    高→中→低   ↓
         黄金标准 → 模型评估和微调
```

### 6️⃣ 指标追踪和分析

✅ **SQLite数据库**:
- 自动时间序列存储
- 支持复杂查询
- 性能优化索引

✅ **分析功能**:
- 聚合统计 (mean, median, p95, p99)
- 模型对比
- 按语言分组
- 时间序列导出
- 异常检测

## 💡 最佳实践

1. **采样策略选择**
   - 一般场景: 使用 `STRATIFIED`（分层采样）
   - 高风险场景: 使用 `RISK_AWARE`（风险感知）
   - 高频场景: 使用 `ADAPTIVE`（自适应）

2. **阈值设置**
   - 高标准: 准确率 > 95%, 一致性 > 90%
   - 中标准: 准确率 > 85%, 一致性 > 80% (推荐)
   - 低标准: 准确率 > 75%, 一致性 > 70%

3. **评估间隔**
   - 高频数据: 每1-5分钟评估一次
   - 中频数据: 每5-15分钟评估一次
   - 低频数据: 每15-60分钟评估一次

4. **标注优先级**
   - priority=2: 一致性 < 50% (立即处理)
   - priority=1: 一致性 50-70% (尽快处理)
   - priority=0: 其他 (常规处理)
