## 🔧 调试过程记录

### 发现和修复的问题

#### 问题1: models.py 中的 dataclass 定义错误
**症状**: `TypeError: non-default argument 'source_text_snippet' follows default argument`
**原因**: RiskAlert 类中非默认参数跟在默认参数后面
**修复**: 为 `source_text_snippet` 和 `source_text_location` 添加默认值

```python
# 修复前
source_text_snippet: str  # ❌ 非默认参数
source_text_location: int  # ❌ 非默认参数
detected_at: datetime = field(default_factory=datetime.now)

# 修复后
source_text_snippet: str = ""  # ✅ 有默认值
source_text_location: int = 0  # ✅ 有默认值
detected_at: datetime = field(default_factory=datetime.now)
```

#### 问题2: preprocessor.py 缺少导入
**症状**: 预处理示例代码中使用了 `datetime`，但未导入
**原因**: 缺少 `from datetime import datetime`
**修复**: 添加必要的导入语句

#### 问题3: 对chromadb 的依赖
**症状**: `ModuleNotFoundError: No module named 'chromadb'`
**原因**: vector_storage.py 需要 chromadb，但测试不依赖此模块
**解决**: 在测试脚本中跳过 VectorStore 导入，使用模拟分析器
